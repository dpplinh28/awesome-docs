# Awesome Docs

Welcome to Awesome Docs — the place where I collect and organize everything I learn, research, and want to remember about technical documentation and developer tooling.

## What this site is for

This site is a personal knowledge base for:

- technical documentation patterns
- research notes
- best practices
- examples and guides
- ideas I want to revisit later

It is not just a project README. It is a living library of documentation knowledge that grows with every new topic.

## What you will find here

- notes from learning sessions and research
- documented workflows and tools
- summaries of technologies and concepts
- practical how-tos and reference material

## How to use this site

- Browse the sidebar to see categories and topic pages.
- Open any page to read a focused note or tutorial.
- Add new documentation pages for things you learn and want to keep.

## Why this matters

Collecting docs like this helps you:

- remember important concepts
- compare tools and approaches
- share what you learn with others
- build a searchable knowledge hub
